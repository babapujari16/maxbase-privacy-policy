Privacy Policy

Last updated: June 2025

This app is a personal tool created to help players of Clash of Clans track upgrade costs and progress.

1. No Personal Data Collected

This app does not collect, store, or share any personally identifiable information (PII).

2. Player Tag Usage

Player tags entered by users are used solely to fetch public game data from Supercell’s official API. This data is not saved, shared, or used for any other purpose.

3. Internet Access

Internet is only used to fetch game-related data via the Supercell API. No tracking, ads, or analytics are currently used.

4. Data Security

The app does not transmit or store any data on external servers. All operations are performed locally or via Supercell’s API with secure HTTPS calls.

5. Third-Party Services

This app uses the official Clash of Clans API provided by Supercell. You can read Supercell’s privacy policy here: https://supercell.com/en/privacy-policy/

6. Children’s Privacy

This app is not directed to children under 13. No personal data is collected knowingly from children.

7. Changes to This Policy

This privacy policy may be updated in the future. Any changes will be clearly reflected in the app update description and within the app.

8. Contact

For questions, contact the developer at: cocmaxbasemonitor@gmail.com

Use of Game Assets

This app may use visual assets from Clash of Clans (such as building or upgrade icons) solely for fan content purposes. These assets are owned by Supercell. Their use complies with Supercell’s Fan Content Policy. This app is not endorsed, sponsored, or approved by Supercell.
